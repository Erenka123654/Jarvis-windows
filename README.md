# 🤖 Jarvis AI Assistant

Modern ve gelişmiş bir Python tabanlı sesli asistan sistemi.

## 🚀 Özellikler

- 🎤 Gerçek zamanlı ses algılama
- 🧠 Sesli komut sistemi
- 🌐 Web tabanlı modern arayüz
- 🔊 Yazıyı sese dönüştürme (TTS)
- 📂 Uygulama açma desteği
- 🔍 Web araması yapabilme
- ⚡ Hızlı ve optimize edilmiş yapı
- 🖥️ Windows uyumlu

---

## 🛠️ Kullanılan Teknolojiler

- Python
- Flask
- SpeechRecognition
- Pyttsx3
- HTML/CSS/JavaScript

---

## 📦 Kurulum

Projeyi klonlayın:

```bash
git clone https://github.com/KULLANICI_ADIN/jarvis-ai.git
```

Klasöre girin:

```bash
cd jarvis-ai
```

Gerekli kütüphaneleri yükleyin:

```bash
pip install -r requirements.txt
```

Projeyi çalıştırın:

```bash
python app.py
```
